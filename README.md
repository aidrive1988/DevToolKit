# ⚡ DevToolKit

A blazing-fast suite of free online developer tools built with [AstroJS](https://astro.build). All tools run 100% client-side — your data never leaves your browser.

## 🛠 Tools

| Tool | Description |
|------|-------------|
| **JSON Formatter & Validator** | Format, beautify, minify, and validate JSON data |
| **Base64 Encoder & Decoder** | Encode and decode Base64 strings with UTF-8 support |
| **URL Encoder & Decoder** | Encode/decode URLs with encodeURI and encodeURIComponent |
| **Color Converter** | Convert colors between HEX, RGB, and HSL with live preview |
| **Lorem Ipsum Generator** | Generate placeholder text in paragraphs, sentences, or words |
| **Word & Character Counter** | Real-time word, character, sentence counting with reading time |

## 🚀 Tech Stack

- **Framework**: [AstroJS](https://astro.build) — Zero JS by default, blazing fast
- **Hosting**: [Cloudflare Pages](https://pages.cloudflare.com) — Global CDN, free tier
- **Styling**: Vanilla CSS with custom design system
- **Processing**: 100% client-side JavaScript

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🏗 Project Structure

```
src/
├── components/     # Reusable Astro components
├── layouts/        # Page layouts (Base, Tool)
├── pages/          # All pages and tool pages
│   └── tools/      # Individual tool pages
└── styles/         # Global CSS design system
```
